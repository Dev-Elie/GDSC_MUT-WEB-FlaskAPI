# GDSC_MUT-WEB-Series

### Working with Virtual environments

#### 1. Creating a virtualenv
                    
**Windows**
          
```bash
py -3 -m venv venv
```
          
**macOS/Linux**
          
```bash
python3 -m venv venv

```

#### 2. Activating the environment
          
**Windows** 

```venv\Scripts\activate```
          
**macOS/Linux**

```. venv/bin/activate```
or
```source venv/bin/activate```
